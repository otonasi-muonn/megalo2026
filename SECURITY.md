# セキュリティポリシー (Security Policy)

## サポート対象について

当プロジェクトは個人開発のため、セキュリティアップデートやバグ修正は**常に最新のバージョン（最新のコミット）に対してのみ**行われます。古いバージョンへのバックポート（遡っての修正）は提供しておりません。

## 脆弱性の報告方法

セキュリティ上の深刻な脆弱性を発見した場合は、**公開のIssueで報告しないでください**（悪意のあるユーザーに利用されるのを防ぐため）。
