# 行動規範 (Code of Conduct)

私たちのプロジェクトに興味を持っていただき、ありがとうございます。
私たちは、誰もが参加しやすく、オープンで歓迎される環境を作ることを目指しています。

## 🟢 期待される行動

私たちコミュニティのメンバーおよびコントリビューターは、プロジェクトをより良くするために以下の行動を実践します：

* 歓迎され、包括的な言葉を使用する
* 異なる視点や経験を尊重し、耳を傾ける
* 建設的なフィードバックを優雅に受け入れ、提供する
* 個人ではなく、プロジェクトとコミュニティ全体にとって最善のことに焦点を当てる
* 他のコミュニティメンバーに対して共感と思いやりを示す

## 🔴 受け入れられない行動

コミュニティを健全に保つため、以下の行動は容認されません：

* 性的、差別的、または不適切な言葉や画像の使用
* 侮辱的・軽蔑的なコメント、および個人攻撃（荒らし行為）
* 公的または私的なハラスメント（嫌がらせ）
* 相手の明示的な許可なく、住所やメールアドレスなどの個人情報を公開すること
* その他、プロフェッショナルな環境において不適切とみなされるあらゆる行為

## 🤝 適用範囲

この行動規範は、プロジェクト内のすべてのスペース（Issue、Pull Request、Discussionsなど）、および個人がプロジェクトやコミュニティを代表して発言する公共の場において適用されます。

## 📩 報告と執行

不適切な行動、ハラスメント、またはこの行動規範に違反する行為を見かけた・受けた場合は、以下の連絡先までご報告ください。

**連絡先**: [Xアカウント](https://x.com/otonasi_muonn)

すべての報告は機密情報として扱われます。プロジェクトのメンテナー（管理者）は報告を真摯に受け止め、状況に応じて、警告、コメントの削除、プロジェクトへの参加ブロックなど、適切かつ公正な対応を行います。
